If you have an LDAP server, you can synchronize user data stored in LDAP at any time when using Casdoor.

After logging in, the `admin` user can click LDAP through the tab bar or directly access `host/ldaps` to add the configuration of LDAP server user information synchronization.

LDAP Sync will synchronized the following user data from the specified LDAP server and converted to Casdoor user data.

- `uidNumber`
- `uid`
- `cn`
- `gidNumber`
- `mail`
- `email`
- `emailAddress`
- `telephoneNumber`
- `mobile`
- `mobileTelephoneNumber`
- `registeredAddress`
- `postalAddress`
